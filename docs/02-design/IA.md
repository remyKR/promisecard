# PromiseCard Information Architecture (IA)

> 작성일: 2025-02-01
> 버전: v1.0

---

## 1. 사이트맵 (1 Depth)

```
promisecard.vn
│
├── 모바일 청첩장 (/) ← 랜딩페이지
│
├── 고객후기 (/reviews)
│
├── FAQ (/faq)
│
├── 고객센터 (/support)
│
└── 마이페이지 (/my) ← 로그인 시
    또는
    로그인/회원가입 (/login, /signup) ← 비로그인 시
```

---

## 2. 네비게이션 상태

### 비로그인 상태

```
[Logo] [모바일 청첩장] [고객후기] [FAQ] [고객센터] [로그인/회원가입]
```

### 로그인 상태

```
[Logo] [모바일 청첩장] [고객후기] [FAQ] [고객센터] [마이페이지]
```
