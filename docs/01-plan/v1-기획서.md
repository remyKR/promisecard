# PromiseCard v1 기획서

> 작성일: 2025-02-01
> 버전: v1.0

---

## 1. 프로젝트 개요

### 1.1 목적
베트남 현지에서 결혼을 준비하는 커플을 위한 모바일 청첩장 서비스

### 1.2 핵심 가치
- **간편함**: 5분 만에 청첩장 완성
- **모바일 최적화**: 카카오톡, Zalo로 즉시 공유
- **현지화**: 베트남어/한국어 지원
- **비용 절감**: 종이 청첩장 대비 저렴

---

## 2. 타겟 사용자

### 2.1 주요 타겟
- 베트남에서 결혼하는 한국인 커플
- 베트남인 커플

### 2.2 사용자 페르소나
| 항목 | 내용 |
|------|------|
| 이름 | 김민수 & 응우옌 티 란 |
| 나이 | 30대 초반 |
| 상황 | 베트남 호치민에서 결혼 예정 |
| 니즈 | 양가 부모님께 쉽게 공유할 청첩장 |
| 페인포인트 | 종이 청첩장 국제배송 비용/시간 |

---

## 3. 기능 범위

### 3.1 v1 MVP 기능 (필수)

| 기능 | 우선순위 | 설명 |
|------|----------|------|
| 청첩장 생성 | P0 | 템플릿 선택, 정보 입력 |
| 미리보기 | P0 | 실시간 미리보기 |
| 공유 링크 | P0 | 고유 URL 생성 |
| 템플릿 | P0 | 최소 3종 |

### 3.2 v1.1 기능 (권장)

| 기능 | 우선순위 | 설명 |
|------|----------|------|
| 회원가입/로그인 | P1 | 소셜 로그인 |
| 마이페이지 | P1 | 내 청첩장 관리 |
| 갤러리 | P1 | 커플 사진 업로드 |
| 지도 | P1 | 결혼식 장소 안내 |

### 3.3 v2 기능 (향후)

| 기능 | 우선순위 | 설명 |
|------|----------|------|
| RSVP | P2 | 참석 여부 확인 |
| 방명록 | P2 | 축하 메시지 |
| 축의금 안내 | P2 | 계좌 정보 |
| 다국어 | P2 | 영어 추가 |

## 4. 비기능 요구사항

### 4.1 성능
- 첫 페이지 로딩: 3초 이내
- 이미지 최적화: WebP 포맷

### 4.2 보안
- HTTPS 필수
- 개인정보 암호화

### 4.3 접근성
- 모바일 반응형 (375px ~ 768px)
- iOS Safari, Android Chrome 지원
