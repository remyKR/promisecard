# PromiseCard

> 베트남 현지 맞춤형 모바일 청첩장 서비스

베트남에서 결혼을 준비하는 커플을 위한 디지털 청첩장 플랫폼입니다.
누구나 쉽게 아름다운 청첩장을 만들고, 링크 하나로 손쉽게 공유할 수 있습니다.

---

## 핵심 가치

| 가치 | 설명 |
|------|------|
| **간편함** | 템플릿 선택 → 정보 입력 → 완성. 5분이면 OK |
| **모바일 최적화** | 카카오톡, Zalo 등 메신저로 바로 공유 |
| **현지화** | 베트남어/한국어 지원, 현지 문화 반영 |
| **비용 절감** | 종이 청첩장 대비 저렴한 가격 |

---

## 주요 기능

### 청첩장 제작
- 다양한 디자인 템플릿 제공
- 사진/동영상 업로드
- 결혼식 정보 입력 (일시, 장소, 지도)
- 실시간 미리보기
- 링크 생성 및 공유

### 부가 기능
- 참석 여부 확인 (RSVP)
- 축의금 계좌 안내
- 갤러리 (커플 사진 모음)
- 방명록

---

## 사용자 흐름

```
[홈페이지 방문]
       ↓
[청첩장 제작하기 클릭]
       ↓
[템플릿 선택] → [정보 입력] → [미리보기]
       ↓
[저장 & 공유 링크 생성]
       ↓
[메신저/SNS로 공유]
```

## 기술 스택

> 바이브 코딩 최적화 (Tier 1 언어 기반)

| 분류 | 기술 | 선정 이유 |
|------|------|----------|
| **Framework** | Next.js 14 (App Router) | SEO, SSR/SSG, 모바일 최적화 |
| **Language** | TypeScript | 타입 안정성, AI 코드 생성 정확도 |
| **Styling** | Tailwind CSS | 빠른 UI 개발, 반응형 |
| **Backend** | Supabase | 인증/DB/스토리지 올인원 BaaS |
| **Database** | PostgreSQL (Supabase) | SQL 기반, AI 이해도 높음 |
| **Storage** | Supabase Storage + Cloudinary | 이미지/영상 업로드 및 최적화 |
| **Maps** | Google Maps API | 결혼식 장소 안내 |
| **Deployment** | Vercel | Next.js 최적화, 자동 HTTPS |

### 아키텍처

```
[사용자]
    ↓
[Vercel - Next.js]
    ↓
[Supabase]
 ├─ Auth (로그인)
 ├─ Database (청첩장 데이터)
 └─ Storage (이미지/영상)
    ↓
[Cloudinary] (이미지 최적화)
```

### 예상 비용 (MVP 단계)

| 서비스 | 무료 플랜 | 비고 |
|--------|----------|------|
| Vercel | 월 100GB 대역폭 | 초기 충분 |
| Supabase | 500MB DB, 1GB 스토리지 | 초기 충분 |
| Cloudinary | 월 25GB 변환 | 초기 충분 |

> MVP 단계에서는 **무료**로 운영 가능
